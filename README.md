# Gender Pay Gap Visualizer

## Overview
A Java-based desktop application that analyzes and visualizes gender pay disparities across different occupations using real-world data.

This tool allows users to search for job titles and compare male vs female average salaries, helping highlight income inequality in a simple and interactive way.


## Features
- Search for job titles
- Display job descriptions
- Compare male vs female average salaries
- Visual progress bars for salary comparison
- Handles missing or inconsistent data
- Case-insensitive and trimmed input matching


## Tech Stack
- Java
- Java Swing (GUI)
- CSV data processing


## How to Run
1. Clone the repository  
2. Open in VS Code  
3. Run:

javac *.java
java MainWindow

## Data Source
U.S. Bureau of Labor Statistics (BLS)
